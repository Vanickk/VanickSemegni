October 2023

Online Thrift Store - 3rd year Software Engineering Project:

Applying agile methodologies and version control, we built an online clothing store demonstrating key software engineering principles. My main task was to build a model that could suggest prices based on the items condition, brand, size ect. Used a node module called "ml-regression-multivariate-linear" to train models based on data such as clothing brand, size and condition ect. The final models were then exported as json files and implemented in Node.js to suggest the price of clothiing based on the above mentionshed attributes. Although the model was not very accurate, the process of implementing the model gave me a better understading of client and serverside communication.
